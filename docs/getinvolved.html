How to contribute?


Requirements

Install NodeJS

curl -sL https://deb.nodesource.com/setup_4.x | sudo -E bash -
sudo apt-get install -y nodejs

Suggestion

Use a Google Chrome Web Browser version (better to debug websockets comm)

Setup environment

Clone the repo

git clone https://github.com/EESC-LabRoM/rosweb.git

Clone submodules

git submodule init
git submodule update

Instal NodeJS dependencies

npm install

Install Gulp globally

sudo npm install -g gulp-cli

Download TypeScript definition files

gulp install

Build

gulp build

Run

gulp start

Tip: While developing, to keep your local server updated, use the watch task:

gulp watch

Did you have any problem following this instructions?

Please, open an issue in our repository, we'll try to fix it as soon as possible!
