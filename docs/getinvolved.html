How to contribute?


Requirements

Install NodeJS

curl -sL https://deb.nodesource.com/setup_4.x | sudo -E bash -
sudo apt-get install -y nodejs

Suggestion

Install Google Chrome Web Browser (better to debug websockets comm)

wget -q -O - https://dl-ssl.google.com/linux/linux_signing_key.pub | sudo apt-key add -
sudo sh -c 'echo "deb [arch=amd64] http://dl.google.com/linux/chrome/deb/ stable main" >> /etc/apt/sources.list.d/google-chrome.list'
sudo apt-get update
sudo apt-get install google-chrome-stable

Setup environment

Clone the repo

git clone https://github.com/EESC-LabRoM/rosweb.git

Clone submodules

git submodule init
git submodule update

Instal NodeJS dependencies

npm install

Install Gulp globally

sudo npm install -g gulp-cli

Download TypeScript definition files

gulp install

Build

gulp build

Run

gulp start

Tip: While developing, to keep your local server updated, use the watch task:

gulp watch
